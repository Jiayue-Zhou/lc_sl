```java
class Solution {
    public boolean isSubsequence(String s, String t) {
        // s length
        int sl = s.length();
        // t length
        int tl = t.length();
        // s pointer index
        int sp = 0;
        // t pointer index
        int tp = 0;
        while (sp < sl && tp < tl) {
            // s char
            char sc = s.charAt(sp);
            char tc = t.charAt(tp);
            if (sc == tc) {
                sp++;
                tp++;
            }
            else {
                tp++;
            }
        }
        if (sp == sl) {
            return true;
        }
        return false;
    }
}
```

Time complexity: O(T)  // size of the String t  
Space complexity: O(1)
