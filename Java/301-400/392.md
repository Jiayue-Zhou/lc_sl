```java
class Solution {
    public boolean isSubsequence(String s, String t) {
        int m = s.length();
        int n = t.length();
        int p1 = 0;
        int p2 = 0;
        while (p1 < m && p2 < n) {
            if (s.charAt(p1) == t.charAt(p2)) {
                p1++;
            }
            p2++;
        }
        return p1 == m;
    }
}
```

Time complexity: O(T)  // Length of the String t  
Space complexity: O(1)
