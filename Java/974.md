```java
class Solution {
    public int subarraysDivByK(int[] nums, int k) {
        int n = nums.length;
        int[] map = new int[k];
        int reminder = 0;
        int ans = 0;
        map[0] = 1;
        for (int i = 0; i < n; ++i) {
            reminder = (reminder + nums[i] % k + k) % k;
            ans += map[reminder];
            map[reminder]++;
        }
        return ans;
    }
}
```

Time complexity: O(n)  
Space complexity: O(k)
