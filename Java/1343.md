```java
class Solution {
    public int numOfSubarrays(int[] arr, int k, int threshold) {
        int n = arr.length;
        int thresVal = threshold * k;
        int left = 0;
        int sum = 0;
        for (int i = 0; i < k; ++i) {
            sum += arr[i];
        }
        int ans = (sum >= thresVal) ? 1 : 0;
        for (int i = k; i < n; ++i) {
            sum += arr[i];
            sum -= arr[left];
            left++;
            if (sum >= thresVal) {
                ans++;
            }
        }
        return ans;
    }
}
```

Time complexity: O(n)  
Space complexity: O(1)
