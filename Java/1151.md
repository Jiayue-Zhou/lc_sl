```java
class Solution {
    // Sliding windows. Counting the number of ones, and find a window of size nums1, with mininum zeros in it.
    // It means if we replace all the zeros in this window, we will get the mininum number of swaps required.
    public int minSwaps(int[] data) {
        int n = data.length;
        int nums1 = 0;
        for (int i = 0; i < n; ++i) {
            nums1 += data[i];
        }
        int left = 0;
        int sum = 0;
        int ans = Integer.MAX_VALUE;
        for (int i = 0; i < nums1; ++i) {
            sum += data[i];
            ans = nums1 - sum;
        }
        for (int i = nums1; i < n; ++i) {
            sum -= data[left++];
            sum += data[i];
            ans = Math.min(ans, nums1 - sum);
        }
        return ans;
    }
}
```

Time complexity: O(n)  
Space complexity: O(1)
